# Set Shortcuts

Use VIA to configure your **MACROPeX MacroBoard**.

### 1. Load the Keyboard Layout
Go to:
https://usevia.app/design

Click **LOAD** and select the **MACROPeX_VIA.json** file.

### 2. Connect Your MACROPeX MacroBoard
Go to:
https://usevia.app/

Click **Authorize New +** in the top-right corner and pair your **MACROPeX MacroBoard**.

### 3. Set Shortcuts
Select any key on the virtual keyboard and assign your preferred shortcut, macro, or function. The changes are saved automatically to your MACROPeX MacroBoard.

# Steps to Update the Firmware

### 1. Connect Your MACROPeX MacroBoard
Connect the MACROPeX MacroBoard to your computer using a USB cable.

### 2. Enter Bootloader Mode
Double-press the **RESET** button on the MacroBoard.

### 3. Open the USB Drive
A new USB drive will appear on your computer.

### 4. Update the Firmware
Copy and paste the **MACROPeX.uf2** firmware file into the root (main) directory of the USB drive.

### 5. Finish
Wait a few seconds for the firmware to install. The MACROPeX MacroBoard will restart automatically with the updated firmware.
